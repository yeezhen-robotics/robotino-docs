This documentation will contain all important documentation regarding this PC's setup,
usage as well as any extra extensions made to the underlying ROS architecture. 
All of the code has been tested on the Robotino 3. The documentation will cover the 
expected set up process for the software of the Robotino and the relevant IP addresses 
required to interface with the LIDAR and the Robotino.
