Robotino Documentation

By: Yee Zhen | Last Updated: 16/04/2026 08:36
Tested on/System Default Ubuntu System: Ubuntu 22.04 LTS (Jammy Jellyfish)

This documentation will contain all important documentation regarding this PC’s setup, usage as well as any extra extensions made to the underlying ROS architecture. All of the code has been tested on the Robotino 3. The documentation will cover the expected set up process for the software of the Robotino and the relevant IP addresses required to interface with the LIDAR and the Robotino.
