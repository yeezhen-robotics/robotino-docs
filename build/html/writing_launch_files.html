Writing Launch Files

Launch files allow multiple ROS nodes to be set up simultaneously. Workarounds have been found by using nodes such as tmux, but this is a guide containing everything learned about launch files.

Most launch file queries can be answered by ChatGPT (normal outline), but more nuanced implementations have to be custom made so that the parts are functional.

This link guides you through different ways nodes can be written so that launch files can launch nodes and other launch files:

https://robotics.stackexchange.com/questions/89429/ros2-include-a-launch-file-from-a-launch-file
